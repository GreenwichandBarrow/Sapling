## Voice + Content Rules

- Educational events reward light prep, not heavy — do NOT force a target list if the event is a panel
- Networking events reward named-target prep — intro lines must reference THEIR work, not G&B's ask
- Per `feedback_know_your_audience`: always tailor to platform + context
- Per `feedback_conferences_must_have_owners`: flag if the event is intermediary/vendor-heavy vs owner-attending
- Per `feedback_no_evening_conferences`: flag timing; daytime-only
- Per `feedback_ignore_one_hanover`: skip One Hanover Happy Hours
- No em dashes
- No HoldCo language in peer-searcher rooms (WSN, Mid-Search Summit, etc.)

## Data Sources

- Event website / registration confirmation
- Speaker bios from event site
- Attendee list if available (request from conference-discovery skill)
- LinkedIn for each panelist / named target
- Attio for prior-touch lookup on all named attendees
