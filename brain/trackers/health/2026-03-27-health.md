## Launchd Permission Error Detail

All 4 failing jobs show the same root cause:
```
shell-init: error retrieving current directory: getcwd: cannot access parent directories: Operation not permitted
/bin/bash: /Users/kaycschneider/Documents/AI Operations/scripts/run-skill.sh: Operation not permitted
```

This is a macOS Full Disk Access issue. The launchd jobs execute under a restricted shell context that lacks permission to access the working directory. Fix: grant Full Disk Access to `/bin/bash` (or the wrapper) in System Settings > Privacy & Security > Full Disk Access, or restructure `run-skill.sh` to cd to an accessible path first.
