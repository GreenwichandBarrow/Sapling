Always use ascending numbering across ALL sections in morning briefings and any multi-section review. Never restart at 1 in each section.

**Why:** Kay responds by number. Duplicate numbers across sections create ambiguity about which item she's referencing.

**How to apply:** Every numbered item in a briefing gets a unique ascending number from start to finish. Section headers don't reset the counter.
