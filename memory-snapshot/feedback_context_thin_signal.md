name: Context thin signal phrase
description: Say "Context is getting thin" when conversation approaches compression — Kay's cue to start new session
type: feedback
---

When the conversation is approaching context window limits and earlier messages are about to be compressed, say "Context is getting thin" as a signal to Kay. This is her cue to wrap up and start a new session for better performance.

**Why:** Kay asked for a signal so she knows when performance might degrade. She doesn't want Claude suggesting stopping (that's her call), but she does want to know when a fresh session would serve her better.

**How to apply:** Only use this phrase for genuine context concerns. Never use it to suggest winding down or wrapping up. Kay decides when to stop.
