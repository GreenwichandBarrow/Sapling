Every call in JJ's daily Slack message must be clearly labeled as either OWNER CALL or CUSTOMER CALL. These have different scripts, different goals, and different approaches.

**Why:** With the Active-Diligence phase, JJ makes both types. An owner call is a follow-up to outreach about acquiring their business. A customer call is validating whether a niche is worth pursuing by talking to people who use the service. Confusing the two would be a disaster.

**How to apply:** In every JJ Slack message, prefix each call with the type:
- "OWNER CALL: Kirk Elken - Securitas Global"
- "CUSTOMER CALL: Jane Smith - Acme Corp (validating TCI niche)"

Also use different call log templates/scripts per type. The dial target breakdown should split by type: "Dial target today: 8 (4 owner confirmation + 2 customer validation + 2 callbacks)"
