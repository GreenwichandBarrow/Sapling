---
name: Never create duplicate Superhuman drafts
description: Cannot delete drafts via CLI, so creating a new draft with updated subject/body creates duplicates Kay has to clean up manually
type: feedback
---

Superhuman CLI can only CREATE drafts, not update or delete them. When updating subject lines or body text, the old draft remains. This creates duplicates Kay has to manually clean up.

**Why:** Kay had to clean up duplicate drafts twice on 3/28. Frustrating and wastes her time.

**How to apply:** Before creating a new draft for the same recipient, warn Kay that the old draft will remain and she'll need to delete it manually. Never batch-create replacement drafts without confirming cleanup plan first. If possible, get the content right the FIRST time to avoid needing replacements.
