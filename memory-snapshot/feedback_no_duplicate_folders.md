---
name: Never create duplicate Drive folders
description: Always search for existing folders before creating new ones on Google Drive
type: feedback
---

Never create a new Google Drive folder without first searching for an existing one with the same name. If Kay created a folder, it already exists — find it, don't duplicate it.

**Why:** Created a duplicate CLIENT VALIDATION CALLS folder instead of finding the one Kay had already made. This caused confusion about where the docs actually were.

**How to apply:**
- Before any `gog drive` folder creation, search for the folder name first
- If a folder exists with that name, use it
- If multiple folders exist, ask Kay which is the right one
- Only create a new folder if search confirms it doesn't exist
