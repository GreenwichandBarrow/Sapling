Never use "Kay" in one-pagers, scorecards, or any professional deliverable. Use "G&B" or "Greenwich & Barrow" instead.

**Why:** These documents are shared with the analyst, investors, and potentially external parties. "Kay's network" reads as informal/personal. "G&B network" reads as professional.

**How to apply:** Any reference to Kay's background, network, connections, or right-to-win should use "G&B" — e.g., "G&B network includes..." not "Kay's network includes...", "G&B luxury background" not "Kay's Chanel background."
