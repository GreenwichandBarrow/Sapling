Ask Kay one question at a time. Never batch 3-5 questions in a single message.

**Why:** Kay flagged that batched questions get lost in the thread. She can't scroll back and loses track of what was asked.

**How to apply:** When you have multiple questions, present one, wait for the answer, then ask the next. If you have context that helps answer without asking, use it instead of adding more questions.
