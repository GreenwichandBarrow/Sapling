Every target recommendation needs a brief "why this target" rationale. Makes Kay's review faster and builds the feedback loop.

**Why:** Anacapa article insight — agents should explain step-by-step reasoning alongside conclusions. Without reasoning, Kay can't give precise feedback on what's wrong.

**How to apply:** Add a "Why" column or note to every target sheet row. Target-discovery sub-agents must include reasoning per target. Niche-intelligence must include reasoning per niche candidate.
