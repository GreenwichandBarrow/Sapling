Superhuman has Kay's full signature built in. When creating drafts, only sign off with:

Very best,
Kay

Do NOT include title, email, website, or confidentiality notice. Superhuman appends all of that.

**Why:** Duplicate signatures look unprofessional.

**How to apply:** Every Superhuman draft created via superhuman-cli.
