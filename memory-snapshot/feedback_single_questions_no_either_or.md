Don't ask a question and its opposite in the same string (e.g., "Do you want X, or should we do Y instead?"). Multiple questions in one message is fine — just don't frame them as A-or-B choices.

**Why:** Kay said "don't ask a question and also the opposite of that question in the same string. It makes it very hard to answer." The problem is the either/or framing, not the quantity of questions.

**How to apply:** State what you think the answer is. If wrong, Kay will redirect. Don't present competing options in one sentence.
