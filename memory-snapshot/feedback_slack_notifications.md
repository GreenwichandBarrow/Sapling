name: Slack notifications must include actionable links
description: Slack workflow notifications should always include the relevant Google Doc/Drive link so Kay can act directly from the notification
type: feedback
---

Slack notifications for completed deliverables must always include the Google Doc or Drive link — not just a text confirmation.

**Why:** Kay acts on Slack notifications directly (e.g., opening a brief on her phone before a meeting). A notification without a link requires extra steps to find the file.

**How to apply:** Any skill that sends a Slack webhook notification for a completed deliverable (meeting briefs, niche reports, etc.) must include the Google Doc/Drive URL in the message body. Never send a "done" notification without the link.
