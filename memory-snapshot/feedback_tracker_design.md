Don't split tracker metrics by role (JJ vs Kay). Track the search funnel as a whole — roles change, the funnel doesn't.

**Why:** JJ's scope is shifting (may just do cold calls now). Role-based splits are fragile and create maintenance burden.

**How to apply:** Any tracker, report, or dashboard should organize by funnel stage (sourcing → qualification → conversion → pipeline health), not by who does the work. The 4 key metrics (NDAs, financials, LOIs submitted, LOIs signed) always go first — everything else is diagnostic explaining whether the machine is feeding those numbers.
