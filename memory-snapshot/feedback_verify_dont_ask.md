Don't ask Kay to confirm whether changes were made to trackers or sheets. Read the sheet yourself and verify.

**Why:** Kay shouldn't be confirming work that the system can verify autonomously. It wastes her time.

**How to apply:** When changes are expected on a Google Sheet or tracker, read it via `gog sheets get` to verify. Only surface to Kay if something is wrong or missing.
