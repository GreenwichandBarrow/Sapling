## Updates (2026-03-19 session)

1. **Post-LOI** (NEW) — 7 event-driven phases: deal cabinet setup, parallel DD workstreams (QoE, Legal, Insurance, Operations, Environmental), financing, weekly DD meetings, red flag escalation, purchase agreement, closing. Framework to evolve with first deal.

2. **Inbound Intermediary Deal Flow** — Pipeline-manager detects CIMs/teasers from brokers, lawyers, CPAs in Gmail. Deal-evaluation has fast buy-box screen (pass/fail/insufficient-info). Speed prioritized for intermediary deals.

3. **Owner Call Prep (Repeat Contacts)** — Meeting-brief auto-detects new vs. repeat contacts via Attio Active Deals lookup. Repeat contacts get 8-section template: Relationship Arc, Pipeline Stage, Prior Call Notes, Financials Status, Open Questions, What to Push For, Seller Personality, Red Flags.

4. **SOP v9** — Added daily Granola action item extraction, environment context (Claude Code, Superhuman, Slack, Google Drive), Wednesday niche intel link at 10am, conference prep doc for Saturday night, meeting schedule aligned to Wed/Thu.

5. **Conference Prep Template** — New template in Master Templates folder for Saturday night conference prep docs.
