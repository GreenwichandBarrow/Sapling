Recording tool stack decided 2026-03-14, replacing Fireflies.ai:

- **Attio Call Intelligence** (Pro plan, already active) — video/audio recording for formal meetings. Bot joins meeting visibly. Auto-links recordings to CRM contacts. Used when recording notification is acceptable.
- **Granola** — stealth meeting intelligence. Local audio capture, no bot, no participant notification. Superior summaries, action items, in-person meeting support. Used when stealth is needed.

**Why:** Kay strongly prefers no-bot/stealth recording (liked this about Fireflies). Neither Google Meet nor Attio is stealth. Granola is the only stealth option. Google Workspace upgrade (Business Starter → Standard) was considered but not worth the cost since it's also not stealth.

**How to apply:**
- Formal meetings where recording is expected → Attio
- Sensitive conversations, in-person meetings, quick calls → Granola
- Build glue: Granola summaries → brain/calls/, Granola action items → Motion tasks, Attio recordings auto-log to CRM
