Kay's productivity tool stack (as of March 2026):
- **Motion** — tasks and task-style projects (primary task manager, migrated from Todoist)
- **Google Calendar** — recurring/scheduled items
- **OneNote** — repository-style lists and reference material
- **Obsidian (brain/)** — knowledge graph, entities, call notes, decision traces

Kay finds Motion's built-in AI challenging — prefers Claude to create tasks/projects via API.
When creating Motion items, order matters: workspace → project (if needed) → tasks.
Projects cannot be updated/deleted via API — name carefully on creation.
Motion supports folders in the UI for grouping projects (not available via API).
