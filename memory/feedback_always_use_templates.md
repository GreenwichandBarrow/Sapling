For ALL deliverables (one-pagers, scorecards, any formatted document), ALWAYS clone an existing working file and replace the text. Never build from scratch with openpyxl or python-pptx styling.

**Why:** Building from scratch produces wrong formatting, wrong column layouts, wrong colors. Kay hates blue. The existing files have the correct styling that Kay approved.

**How to apply:**
- One-pagers: Clone Art Advisory pptx (Drive ID: 1GDBgRyNWJ2vGn7niEeqjrAGfyqPRK9eX), swap run text
- Scorecards: Clone Insurance scorecard xlsx, swap cell values in the existing column layout (B-J)
- Never invent formatting. Never add blue. Always match what exists.
- Before uploading, check for existing file in target folder and delete first (no duplicates)
