**Rule:** External-facing G&B copy MUST NOT name a geographic restriction. No "NY / NJ / PA / CT," no "tristate," no "Northeast preferred," no state list. This includes broker outreach emails (Day 0 / Day 5 / Day 12), intermediary outreach to lawyers / CPAs / family offices / lenders / association heads, broker-facing buy-box one-pagers, or anything else a non-investor counterparty might see.

**Why:** G&B's investors view geographic restrictions as a "big nono" to publish externally. Reasoning the operator gave 2026-05-04: signaling geo restriction looks limiting / un-serious, and creates friction if a strong out-of-geo deal does eventually surface and we want to look at it without appearing inconsistent. The geographic preference is a routing filter for our side, not a public criterion.

**Where geography STAYS:**
- Broker buy-box doc in the analyst folder (`brain/context/buy-box-broker-channel.md` + Drive doc `1feNR94YgksrJAEtRYGZOmHQxYe3vaG72Pl0M-bR61M0`) — internal screen for the analyst + deal-aggregator
- deal-aggregator OPPORTUNISTIC channel routing logic
- target-discovery niche-filtering logic
- Internal scorecards and deal-evaluation pipelines
- Personal task tracker and operational sheets

**Where geography MUST NOT appear:**
- Broker outreach Day 0 / Day 5 / Day 12 templates (Brokers + IBs unified)
- Intermediary outreach to Lawyers / CPAs / Family Offices / Lenders / Association Heads
- Conference outreach drafts
- Owner outreach (already silent on geography per `feedback_no_revenue_in_outreach` adjacent rules)
- Any one-pager, buy-box card, or signature block that goes to non-investor recipients
- The G&B website (review and ensure no geo restriction language there)

**How to apply:**
- When drafting any external-facing copy, the buy-box footer lists Independent ownership / Revenue / EBITDA / Margin / Industries — and stops. No geography line.
- If a recipient asks about geographic preferences on a call, the operator can answer directly. Do not pre-publish.
- When deal-aggregator filters for geo, the filter happens silently on our side — the broker's email goes through but we may decline a clean-financials-but-out-of-geo deal in private. This is fine; brokers won't notice the filter.
- Internal docs CAN keep geography. The boundary is the eyeball of the recipient. Internal eyeballs (G&B operator, analyst, future hires, investors): geography in. External eyeballs (brokers, intermediaries, owners, counterparty firms): geography out.

**Caught:** 2026-05-04 broker outreach lock. Initial Brokers + IBs footer drafts included "NY / NJ / PA / CT" line. G&B operator removed: "I hesitate to put geographic into the email when its a big nono for my investors." This memory exists so geography never re-enters external copy.
