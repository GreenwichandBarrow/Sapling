The Insurance Producer License Compliance customer validation call list format is the standard for ALL call list docs going forward. Kay updated it with Avenir font and G&B logo letterhead.

**How to apply:**
- Plain text, no markdown characters (no #, **, ---, -)
- Section headers in caps
- Numbered contacts with plain labeled lines (Contact, Title, Phone, LinkedIn, Location, Why)
- Avenir font, black text, G&B logo centered at top
- Reference the IPLC doc (ID: 113Ma8mFrNoeWwPRtjOjr62Z8yShtemCYGmW-_RmYiJo) as the formatting template when creating new docs
