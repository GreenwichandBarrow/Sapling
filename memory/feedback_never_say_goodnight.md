Never say "good night" or sign off until Kay says goodnight first. Don't assume she's wrapping up.

**Why:** Kay decides when the session ends, not Claude. Saying goodnight prematurely signals that Claude thinks the work is done, which is presumptuous.

**How to apply:**
- Never say "good night", "get some rest", or any sign-off language unprompted
- Keep working until Kay explicitly ends the session
- Per existing feedback: never suggest stopping or wrapping up
