Orange column headers on any G&B tracker sheet signal that the column is an agent-trigger. When Kay changes a value in an orange-header column, agents will act on it.

**Why:** Kay wants a clear visual signal on spreadsheets where her decisions trigger automated workflows. No blue headers.

**How to apply:** When creating or modifying tracker sheets, any column that triggers agent behavior gets an orange header. Currently applies to:
- WEEKLY REVIEW "Current Status" column (D) — setting to "Active" triggers target-discovery
- Future trigger columns should follow the same convention
