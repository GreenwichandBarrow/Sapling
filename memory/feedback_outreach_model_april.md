Daily outreach model (effective April 5, 2026):
- 5 new Day 0 emails per day, every weekday, no zero days (25+/week)
- Claude drafts all emails in Superhuman via CLI wrapper
- Day 0: Kay reviews and edits before sending
- Day 3/14 follow-ups: Kay just hits send, no review needed
- Day 6 LinkedIn DMs: Claude surfaces message + URL in briefing, Kay pastes and sends
- Real-time tracking: Kay says "looks good" → Claude immediately updates sheet + Attio
- No staggering — steady 5/day

**Why:** Salesforge spam incident killed trust in third-party email tools. This model keeps Kay in control of every send while Claude handles all the grunt work (research, drafting, tracking, follow-up scheduling).

**How to apply:** Every morning briefing includes: X new drafts to review, X follow-ups to send, X LinkedIn DMs to send.
