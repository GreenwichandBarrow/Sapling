New umbrella niche created Apr 9 for DealsX's 3rd slot (replacing Fractional CFO). The thesis: industries with multi-workflow operations (retail + repair + compliance + unique asset inventory) where generic software fails.

**Pattern:** High-value assets needing tracking, compliance documentation, service/repair workflows, and wealthy clients expecting white-glove operations. Software is fragmented, bootstrapped, ripe for consolidation.

**Sub-verticals (11):**
- Jewelry Retail & Repair Management Software (70 vendors on Capterra)
- Marina & Boatyard Management Software (69 vendors on Capterra)
- Art Gallery & Collection Management Software (30+)
- Private Club Management Software (20-30)
- Equestrian Stable & Farm Management Software (15-20)
- Event Rental Management Software (15-20)
- Wine & Spirits Inventory/Compliance Software (20+)
- Auction House Management Software (12-15)
- Estate/Household Management Software (10+)
- Fine Art Logistics & Shipping Platforms (15+)
- Classic/Collector Car Restoration Shop Software (8-12)

**Excluded:** Firearms (not personal fit), Private Aviation/FBO (investor warned too competitive)

**Total estimated vendor pool:** 300-400+ software companies

**Key insight from Guillermo call:** Jewelry stood out because repair services create unique tracking needs outside typical retail. Same pattern applies across all sub-verticals.

**Why:** Guillermo said the only deal Ashford got close on was software. Software can work with right criteria: deep moat (not easily AI'd), not team-dependent, compliance-driven, searcher fit.
